# Система многослойного моделирования архитектуры

## Установка приложения

1) Распакуйте архив symma.tgz в каталог /opt/symma (или на ваш выбор)
2) Установите docker-образы
3) Запустите приложение из рабочей директории с помощю docker-compose

bash
tar -xvf symma.tgz
docker load --input symma.tar && docker load --input postgres.tar
docker-compose up -d



## Настрока приложения

По умолчанию приложение запускается на порту 80/tcp, пользователь БД mt пароль 123456,
для изменения вышеперечисленных параметров отредактируйте файл .env Имя пользователя БД
и пароль желательно изменить до первого запуска приложения. Если имя пользователя БД и
пароль были изменены, то после запуска приложения эти данные необходимо указать в конфигурации
приложения по адресу http://your.server.name/config (пароль по умолчанию admin)
Далее можно приступать к работе, http://your.server.name/ пользователь/пароль admin/admin
